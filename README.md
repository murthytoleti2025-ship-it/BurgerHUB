# BurgerHub - Food Delivery System

A Flask and MongoDB-based web application for ordering burgers.

## Features
- **Roles:** Admin, Customer, and Delivery.
- **Functions:** Order tracking, menu management, and topping customization.
- **Tech Stack:** Python (Flask), MongoDB.

## How to Run
1. Install dependencies: `pip install flask pymongo`
2. Ensure MongoDB is running on `localhost:27017`.
3. Create a database named `BurgerHub`.
4. Run `python app.py`.
